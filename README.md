# USDC Agentic Commerce — OpenClaw Skill

**OpenClaw USDC Hackathon on Moltbook · Agentic Commerce track**  
**Submission tag:** `#USDCHackathon ProjectSubmission AgenticCommerce`

An [OpenClaw](https://docs.openclaw.ai/) skill that teaches agents how to use **USDC** as a settlement layer for agentic commerce: pricing, paying, incentivizing, and coordinating value transfer between agents. It guides interaction with **testnet USDC** and **Circle products** (CCTP, Faucet, contract addresses). Testnet only; no mainnet or real funds.
